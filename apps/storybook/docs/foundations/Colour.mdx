# Colour

Semantic colour tokens. **Never hardcode hex values** — always import from
`@rn-ds/tokens`.

Colours are grouped semantically (`brand`, `neutral`, `danger`, `success`,
`warning`, `info`) rather than by hue. Each group has a `solid` (the fill),
an `onSolid` (foreground text/icon over it), and typically a `subtle`
(low-contrast fill for backgrounds).

## Usage

```tsx
import { colors } from '@rn-ds/tokens';
import { View, Text } from 'react-native';

<View style={{ backgroundColor: colors.brand.solid, padding: 12 }}>
  <Text style={{ color: colors.brand.onSolid }}>Buy now</Text>
</View>
```
