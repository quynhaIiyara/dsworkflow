# Colour

Colour tokens. **Never hardcode hex values** — always import from
`@rn-ds/tokens`.

The palette mixes surface/ink/hairline semantic keys with named hue keys
(`green`, `red`, `amber`, `blue`, plus `*Pale`/`*Dark`/`*Text` shades) used
across status, feedback, and brand accents.

## Usage

```tsx
import { colors } from '@rn-ds/tokens';
import { View, Text } from 'react-native';

<View style={{ backgroundColor: colors.blue, padding: 12 }}>
  <Text style={{ color: colors.white }}>Buy now</Text>
</View>
```
