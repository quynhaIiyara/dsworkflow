# Product Recommendation

The product-advisory card the agronomist attaches to a completed record.
Product identity + delete affordance on top, quantity stepper in the middle
(bag count + MT conversion below the primary value), and a required
free-text advisory note with a live character counter.

Diverges from `gallery.html #c28` — the current design uses a stepper and
required textarea instead of the collapsible-note pattern in the gallery
snapshot.

**Built from:** `Text` · inline `StepperControl` + `QuietTextarea` + trash
button + icon tile · icons (`PackageIcon`, `TrashIcon`)
